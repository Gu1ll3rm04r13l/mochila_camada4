¿Quién iventó el sistema de control de versión Git y por qué?

La persona que inventó el sistema de control de versión Git fue Linus Torvalds.
Lo hizo porque anterior a su existencia había muchos problemas en el mantenimiento de versiones de aplicaciones cuanto estas tenían un gran número de archivos de código fuente
