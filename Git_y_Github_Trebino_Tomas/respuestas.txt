a. ¿Quién inventó el sistema de control de versión Git y por qué?
   Git fue inventado por Linus Torvalds para llevar registro de los cambios en archivos de computadora y para coordinar el trabajo que varias personas realizan sobre archivos compartidos en un repositorio de código.
b. ¿A Quién pertenece actualmente Github y por qué?
   Git pertenece a la comunidad ya que tiene una licencia publica.
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
   Si, se puede trabajar desde la página web.
