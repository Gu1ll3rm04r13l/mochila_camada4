¿Quién inventó el sistema de control de versión Git y por qué?
 Linus Torvalds fue el inventor de Git. pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

¿A quién pertenece actualmente Github y por qué?
Microsoft compró Github. Microsoft desea volver a ser relevante. Y el camino es el código abierto.
