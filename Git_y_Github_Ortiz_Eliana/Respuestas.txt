a. ¿Quién inventó el sistema de control de versión Git y por qué?

Git fue creado y desarrollado por Linus Torvalds, quien necesitaba una herramienta propia para trabajar en el control de versiones, ya que BitKeeper había dejado de ser ofrecida de forma gratuita.

b. ¿A quién pertenece actualmente Github y por qué?

Github pertenece actualmente a Microsoft, ya que la compañía adoptó una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios. 

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Sí, también se puede trabajar con Github desde su UI, representado en la aplicación para Desktop.
