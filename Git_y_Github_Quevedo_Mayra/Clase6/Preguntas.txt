a) ¿Quién inventó el sistema de control de versión Git y por qué?
b) ¿A quién pertenece actualmente Github y por qué?
c) ¿Hay otra forma que no sea la terminal para trabajar con Github?

a) El creador de Git es Linus Torvalds, quien es también el creador de Linux. 
Su empresa se encontró con un problema al no poder seguir usando su sistema 
de control de revisión BitKeeper y al no conseguir ningún otro que satisficiera
las necesidades de su empresa, creó Git.

b) Actualmente, Github pertenece a la compañía Microsoft. Estos compraron la empresa
con el propósito de erradicar la mala visión que los desarrolladores tienen de ellos,
ya que en 2001 el en ese momento CEO de Microsoft, Steve Ballmer se declaró
abiertamente en contra del código abierto. También es un fuerte motivo,
que su mayor competencia Amazon, tenga un tan rapido crecimiento.

c) Podemos usar Github desde la terminal o desde una gran variedad de interfaces
de usuario con distintas capacidades. Cabe destacar que solamente desde la línea
de comandos podemos utilizar todos los comandos de Git en su totalidad, no así en las interfaces.
